## Project Abstract

This research project focuses on innovating the aluminum recycling process by utilizing machine learning techniques to automatically identify and analyze the formation of intermetallic compounds (IMCs) in recycled aluminum.

The goal is to develop a workflow that can dramatically reduce the manual annotation effort required for large-scale recycling. By using weakly supervised deep learning models, the project aims to provide a solution for industry, supporting a leap in efficiency and sustainability.

This collaborative effort, supported by a consortium of experts from synchrotron facilities, machine learning, and industry, aims to deliver a reusable and transferable tool for volumetric 3D semantic segmentation.
