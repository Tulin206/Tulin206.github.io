## Project Overview

Developed an interactive augmented reality system using HoloLens and tablet integration to visualize and interact with 3D medical imaging data (liver CT), enabling enhanced surgical planning and medical training.

<!--more-->

## Project Objectives

- Create immersive 3D visualization of medical imaging data
- Enable spatial navigation of volumetric data
- Develop gesture-based interaction for medical applications
- Support surgical operation planning with 3D organ models

## Technical Achievements

### HoloLens Application
- **Gesture Interaction**: Developed intuitive gesture-based controls for liver model interaction
- **Region Selection**: Enable surgeons to select specific CT regions for detailed analysis
- **Spatial Tracking**: Real-time tracking of head and hand movements
- **Operation Planning**: Support pre-surgical planning and visualization

### Tablet Integration
- **Complementary Interface**: Tablet serves as control and navigation device
- **Spatial Awareness**: Coordinated interaction between HoloLens and tablet
- **Enhanced UX**: Multi-device workflow for medical professionals

### 3D Visualization
- **Volumetric Data Handling**: Process and render 3D CT liver models
- **Real-time Rendering**: Smooth interaction with complex 3D data
- **Anatomical Accuracy**: Preserve medical imaging integrity in AR environment

## Technologies Used

**Game Engine**: Unity3D (C#)

**VR/AR Platforms**: HoloLens, Mixed Reality

**Development Tools**: Visual Studio, Unity Editor

**3D Graphics**: HLSL shaders, Unity rendering pipeline

**3D Models**: CT image data conversion and mesh generation

## Clinical Applications

- **Surgical Planning**: Pre-operative visualization and route planning
- **Medical Training**: Enhanced learning through immersive visualization
- **Patient Education**: Explain complex anatomy to patients
- **Collaborative Planning**: Multi-surgeon consultation and planning

## Innovation Highlights

- One of the early applications of mixed reality in surgical imaging
- Gesture-based interaction for hands-free operation in clinical settings
- Seamless integration between tablet and HoloLens devices
- Real-time volumetric data rendering in mixed reality
