## Project Overview

Researching and implementing self-supervised deep learning methods to denoise raw tomography detector data (sinograms) for improved image reconstruction in ultrafast electron beam X-ray computed tomography.

<!--more-->

## Research Objectives

- Develop self-supervised methods for sinogram denoising
- Improve image reconstruction quality from noisy detector data
- Establish robust evaluation metrics for tomography data
- Contribute to advancing computational imaging techniques

### Technical Achievements

- SSIM (Structural Similarity Index)
